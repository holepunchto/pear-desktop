# Pear Desktop

> The Pear Runtime Pear Desktop Application

Pear Desktop is the [pear://runtime](pear://runtime) Application.

## Usage

```
pear run pear://runtime
```

## Development

```
git clone https://github.com/holepunchto/pear-desktop
cd pear-desktop
pear run --dev .
```
